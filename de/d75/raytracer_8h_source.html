101/**
102 * Computes the intersection of a line with a plane.
103 *
104 * The first step of this algorithm is to compute v0 and v1. These
105 * are vectors from the plane's reference point to each of the line-points,
106 * respectively.
107 * We then take the dot-products of these
108 * vectors with the plane normal.
109 * We then say that the vectors have a positive sense if the dot-product
110 * is positive and a negative sense if the dot-product is negative.
111 * If the senses are not equal then the line intersects the plane.
112 *
113 * Since the face normal is a normalized vector the dot-product of v0 or v1
114 * will give the projection of the relevant vector along the normal to the
115 * plane. We can use this projection to compute a weight associated with
116 * each vector. This also then allows us to compute the intersection point.
117 *
118 * \param plane_normal The normal associated with the plane
119 * \param plane_point The reference point for the plane
120 * \param line_point_0 The line's initial point
121 * \param line_point_1 The line's destination point
122 * \param intersection_point The point to be populated with the intersection
123 * point
124 * \param weights The weights associated with this intersection
125 *
126 * \return Returns true if the line intersects the plane and false otherwise.
127 *
128 * \author Jan
129 */
130bool CheckPlaneLineIntersect(const Normal& plane_normal,
131 const Vector3& plane_point,
132 const Vector3& line_point_0,
133 const Vector3& line_point_1,
134 Vector3& intersection_point,
135 std::pair<double, double>* weights = nullptr);

137/**

172 * This functions checks the intersection of a plane with a tetrahedron.
