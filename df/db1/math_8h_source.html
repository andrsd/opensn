55/**Sample a Cumulative Distribution Function (CDF) given a probability.
56 *
57 * The supplied vector should contain the upper bin boundary for each
58 * bin and will return the bin associated with the bin that brackets
59 * the supplied probability.
60 *
61 * Example:
62 * Suppose we sample bins 0-9. Suppose also that the probalities for each
63 * bin is as follows:
64 * - 0.1 bin 0
65 * - 0.1 bin 1
66 * - 0.5 bin 5
67 * - 0.3 bin 8
68 *
69 * The CDF for this probability distribution will look like this
70 * - bin 0 = 0.1
71 * - bin 1 = 0.2
72 * - bin 2 = 0.2
73 * - bin 3 = 0.2
74 * - bin 4 = 0.2
75 * - bin 5 = 0.7
76 * - bin 6 = 0.7
77 * - bin 7 = 0.7
78 * - bin 8 = 1.0
79 * - bin 9 = 1.0
80 *
81 * Supplying a random number between 0 and 1 should indicate sampling one
82 * of the bins 0,1,5 or 8. The most inefficient way to do this is to
83 * linearly loop through the cdf and check \f$ cdf_{i-1} \ge \theta < cdf_i \f$.
84 * An optimized version of this sampling would be to perform a recursive
85 * block search which starts with a course view of the cdf and then gradually
86 * refines the view until the final linear search can be performed.*/
87int SampleCDF(double x, std::vector<double> cdf_bin);
